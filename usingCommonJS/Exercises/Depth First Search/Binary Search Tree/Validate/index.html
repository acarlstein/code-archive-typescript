Exercise: Depth First Search: Binary Search Trees: Validate

Write a code that will validate the binary search trees shown below:

Tree A

  • 2
    • 1
    • 3

[2, 1, 3]
True BST

Tree B

  • 1
    • 2
    • 3

[1, 2, 3]
False BST

BST Definition:
All nodes in the left side contains keys with less value than the parent node.
All nodes in the right side contains keys with greater value than the parent node.
Both left and right subtrees must be BST.

Time Complexity Expected: O(N).

Space Complexity Expected: O(1).

Press F12, access the console and refresh the page to run script.

Solution:

Time Complexity: O(N). We visit each node in the tree recursively.

Space Complexity: O(1). Only one extra variable, previousValue, is used.

Explanation:
To validate a BST, we perform an in-order traversal while ensuring that each node is in ascending order. Example:
