Exercise: Breadth First Search: Binary Trees: Symmetric Tree Around Its Center

Write a code that will verify if a binary tree is symmetric around its center.
In other words, verify if the binary tree is a mirror of itself.
Note: The solution can be recursive.

Time Complexity Expected: O(N).

Press F12, access the console and refresh the page to run script.

Solution:

Time Complexity: O(N) = O(N1) + O(N2) where N1 and N2 are the number of nodes of each tree.

Space Complexity: O(N). Each execution of the function is O(1); however, this function is called recursivelly. Therefore it becomes O(N).
