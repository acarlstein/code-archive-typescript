Exercise: String: Find Longest Substring without Repetition

You are given an array of strings.
For each string, you must find the longest substring in which characters are not repeated.

For example, the string "abccc" will produce the longest substring "ab" because the character c is repeated.
The string "abcabcdd" will produce the longest substring "abcd"; therefore, the length of the substring is 4.
The string "ccccc" will produce the longest substring "c"; therefore, the length of the substring is 1.
The string "paakea" will produce the longest substring "ake"; therefore, the length of the substring is 3.
Note: "pake" would be a subsequence but not a longest substring. 

            let strArr : string[] = ["abcabcdd", "ccccc", "paakea"];
            return [4, 1, 3]

Time Complexity Expected: O(N) or better for each string processed

Space Complexity Expected: O(N) or better for each string processed

Press F12, access the console and refresh the page to run script.

Solution:

Time Complexity (to find each substring): O(N). The array is traversed once for each string.

Space Complexity (to find each substring): O(N).

Explanation:
We use a hashmap to store the maximum count of characters for each character (key).
As we move from right to left throw the string, one character at the time, we update the hashmap.
If a character already exists in the hashmap, we increase the counter and move to the next character to the left.
The variable count is used as a pointer of position and counter.
