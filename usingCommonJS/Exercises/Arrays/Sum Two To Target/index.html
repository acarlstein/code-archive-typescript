Exercise: Arrays: Sum Two to Target

You are given an array of numbers.
You are asked to return the indices of two elements that sum up to a specific target.

Note: As a rule, you are not allowed to use a same element twice or more.

I.e: If you have elements [3, 5, 8, 13] and the target is 11 then 3 + 8 = 11; therefore, we return index 0 (for 3) and index 2 (for 8) in an array

            let arr : number[] = [3, 5, 8, 13];
            return [0, 2]; // returning index 0 and 2

Time Complexity Expected: O(N) or better

Space Complexity Expected: O(N) or better

Press F12, access the console and refresh the page to run script.

Solution:

Time Complexity: O(N). The array is traversed once.

Space Complexity: O(N).

  • O(1) to create variables
  • O(N) for the hashmap which will increate linearly due the increase of number of entries
  • O(N) = O(1) + O(N)
