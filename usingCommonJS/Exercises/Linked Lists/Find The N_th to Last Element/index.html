Exercise: Linked List: Find The Nth to Last Element

You are given a linked lists for which you don't know when it ends.
Linked List: [1]->[2]->[3]->...->[N]

You are asked to write a function which allows the user to find the Nth-to-last element of such list.
Example: 

            list = [1]->[2]->[3]->[4]->[5]
            Last element is [5].
            
            When n = 1, output = 4:             
            [1]->[2]->[3]->{4}->[5]
            
            When n = 2, output = 3:
            [1]->[2]->{3}->[4]->[5]

            When n = 3, output = 2:
            [1]->{2}->[3]->[4]->[5]

Time Complexity Expected: O(N).

Space Complexity Expected: O(1).

Press F12, access the console and refresh the page to run script.

Solution:

Time Complexity: O(N).

Space Complexity: O(1).

Solution:
The trick here is to use two node pointers.
First pointer, called current, will take care of tracking the current position in the linked list.
The second pointer, called follower, will begin to follow the current pointer by a distance equals to the Nth position.
Using the example above, if Nth value is 2, this means that recently the follower pointer will begin moving when there is a distance between it and the current pointer.
