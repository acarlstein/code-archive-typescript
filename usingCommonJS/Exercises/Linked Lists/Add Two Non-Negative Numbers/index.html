Exercise: Linked List: Add Two Non-Negative Numbers

You are given two linked lists which are not empty and represent two non-negative numbers.
The digits of these numbers are stored in reverse order. Each node contains a single digit. Write a code that will add two numbers and return a single linked list.

For example,
Linked List 1: [2] -> [3] -> [4]
Linked List 2: [5] -> [6] -> [7]
Linked List Result: [7] -> [9] -> [1]

Note: Please note that 4 + 7 = 11 but we store only one digit, the last digit will be carried to the next summation.
Linked List 1: [4] -> [2]
Linked List 2: [7] -> [6]
Linked List Result: [1] -> [9]
4 + 7 = 11; therefore, we store 1 and the other 1 we do 2 + 6 + 1 = 9

Time Complexity Expected: O(N).

Space Complexity Expected: O(N).

Press F12, access the console and refresh the page to run script.

Solution:

Time Complexity (to find each substring): O(N) = O(N1) + O(N2)
where N1 represent the length of List 1 (L1) and N2 represents the length of List 2 (L2).

Space Complexity (to find each substring): O(N) = O(1) + O(N). O(1) for the variables and o(N) for the result.

Explanation:
The digits are stored in reverse order; therefore, we can traverse both lists and add their values.
The only difference is that when adding two numbers, if we exceed 10, we will only store the last digit.
The other digit will be carried to the next summation. The process will repeat.
