Exercise: Bit Manipulation: Find Number of 1s from Binary Representation of a Number

From the binary representation of a number, we wish to find the number of 1s.

For example: 

            getNumberOfOnes(2) = 1  // 2 = 0b01
            getNumberOfOnes(3) = 2  // 3 = 0b11
            getNumberOfOnes(5) = 2  // 5 = 0b101
            getNumberOfOnes(7) = 3  // 7 = 0b111
            getNumberOfOnes(11) = 3 // 11 = 0b1011

Time Complexity Expected: O(N)

Press F12, access the console and refresh the page to run script.

Solution:

There are two ways.
First, we sum the Last Significant Bit (LSB) which is the last bit to the right.
How we do that? Well, there are two method:

  • Remainder method: Mod the number by two (num % 2).
    If the number is even, we get 0. If the number is odd, we get 1.
    Then, shift to the right by one the number to obtain the next LSB.
  • Bitwise method: Use the AND bitwise to obtain a result where all the bits are zero except the last one (LBS).
    The operation is (num & 1).
    Using the basic AND bitwise table and assuming x is a bit state, we see that:
    Operation Result
    x & 0 0
    x & 1 x
    x & x x
    Therefore, if we have a number in binary, it would work this way:
     0b1010101011
    &
     0b0000000001
     0b0000000001
     0b1010101010
    &
     0b0000000001
     0b0000000000

    Then, shift to the right by one the number to obtain the next LSB.
