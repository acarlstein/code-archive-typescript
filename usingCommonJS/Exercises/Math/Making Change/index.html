Exercise: Math: Making Change

We are given certain amount that represent a currency.
Assume you have an ATM that can use coins.
We are asked to determine the minumum number of coins required to make that exact amount in change.
Note: The coins system that we can use are 1, 5, 10, and 25 cents.

For example: 

            let coinSystem : number[] = [1, 5, 10, 25]; // In Cents
            let amount : number = 32;

Press F12, access the console and refresh the page to run script.

Solution:

Explanation:
The current solution is a recursive one.
In each recursion, will receive a new amount which would be the previous amount minus the biggest coin in the coin system in sequence.
Below is a table that will show the idea behind:

Iteration Amount Current coin Coin counter
1 32 25 1 (25 cents)
2 7 10 1 (No change)
2 7 5 2 (25 + 5)
3 2 1 3 (25 + 5 + 1)
4 1 1 4 (25 + 5 + 1 + 1)
Result: 4 coins are needed.
