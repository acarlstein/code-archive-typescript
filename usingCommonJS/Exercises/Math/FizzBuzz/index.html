Exercise: Math: FizzBuzz

We are provided with a number x and we are asked to iterate from 1 to x.
As we iterate, each number will be checkout.
If the number is divisible by 3, print the word "Fizz".
If the number is divisible by 5, print the word "Buzz".
If the number is divisible by 3 and 5, print the word "FizzBuzz".
Else, just print the number as it is.
Note: The number x cannot be negative.

For example: 

            let x : number = 16;
            1
            2
            Fizz
            4
            Buzz
            ...
            14
            FizzBuzz
            16

Time Complexity Expected: O(N)

Press F12, access the console and refresh the page to run script.

Solution:

Explanation:
To solve this issue, we are going to use the remainder (%) operator.
The remainder operator returns the remainder of two numbers.
It is useful when trying to figure out when a number is odd or even and restrictinv a value to a range.
In our case, we want to restrict a value to 3 and 5, 3, then 5.

Notice that we switched the order in which we test the value by checking first if the value is divisible by 3 and 5.
This is done of purpose because we would never had that statement being true if we would check first if a number is divisible by 3 or by 5.

It's always good to go over your code and check that it does the job correctly.
