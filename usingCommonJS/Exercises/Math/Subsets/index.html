Exercise: Math: Subsets

A set is a well-defined collection of objects whcih normally share a common characteristic.
A subset is a portion of a set which share some common elements.

Assume you are given a set of numbers and you are asked to create a function.
The function should return all possible distintic subsets and ensure that no duplicate subsets are returned. For example: 

            let set : number[] = [1, 2, 3];
            return [[], [1], [2], [3], [1,2], [1,3], [2,3], [1,2,3]];

In this case, the number of subsets should be 2n = 23 = 8 subsets.
Another approach is to analyze this problem as it would be a tree:

  • [ ]
    • [ ]
      • [ ]
        • [ ]
        • [3]
      • [2]
        • [2]
        • [2, 3]
    • [1]
      • [1]
        • [1]
        • [1, 3]
      • [1, 2]
        • [1, 2]
        • [1, 2, 3]

Hint: The solution can be built recursively.

Time Complexity Expected: O(2n)

Space Complexity Expected: O(1)
