Trie trees are also called prefix trees, digital trees and sometimes radix trees.
Trie trees are similar to binary search tree; however, the position of the nodes are defined by association.
Common applications for a trie tree is storying a predictive text or automcomplete dictionary commonly used on devices such as smartphones.
The easy way to understand a prefix tree is to see a visual example of one:

Trie tree (aka. Prefix tree, aka. Digital tree, aka. Radix Tree)

  • root
    • t
      • to
      • te
        • tea
        • ted
        • ten
    • a
    • i
      • in
        • inn

Keys: ["a", "to", "tea", "ted", "ten", "i", "in", "inn"]

For this exercise, you are asked to implement a Trie object that has insert, search, and startsWith methods.
Assume that that all inputs consists of lower case letters from a to z.

Press F12, access the console and refresh the page to run script.

Solution:

Time Complexity: O(N) = O(N1) + O(N2) where N1 and N2 are the number of nodes of each tree.

Space Complexity: O(N). Each execution of the function is O(1); however, this function is called recursivelly. Therefore it becomes O(N).
