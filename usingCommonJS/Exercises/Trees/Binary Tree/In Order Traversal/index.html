Exercise: Trees: Binary Tree: In-Order Traversal

You are given a binary tree.
You must return the In-Order traversal of the node values of that Binary Tree.
Make your code iteratively instead of recursive.

Time Complexity Expected: O(N).

Space Complexity Expected: O(log N).

Press F12, access the console and refresh the page to run script.

Solution:

Time Complexity: O(N). We visit each node in the tree.

Space Complexity: O(log N). Since we are using a stack. We will need a maximum of log N elements (including ancestors elements).

Explanation: In order to do this code run iteratively, we must store all the root elements; therefore, we will use a stack. The idea is to store all te root elements. Then, process the left subtree. Once the left subtree is processed, we will add the root value to the list. Then, we repeat the process to the right subtree.
